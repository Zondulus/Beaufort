**Introduction**

Beaufort is a unified GUI for all of my wind & weather mods built for WindAPI. It is a simple and unobtrusive display with four modes which supply all of the information about the current weather any pilot could need. In addition, it also provides a simple way to add wind via its own integrated controller. Beaufort captures all of the information from any combination of wind & weather mods built for WindAPI in one place.



Beaufort's four modes are:

* Default (DEF): a display with simple descriptions of the current wind, strength, vertical component, and turbulence level


* Exact (EXC): an exact numerical breakdown of the current total wind


* Barb (BRB): a functioning animated meteorological wind barb which symbolically represents the current wind velocity


* Control (CTL): a control panel for creating a wind gust with a specified heading, horizontal speed, and vertical speed



When I first started making mods, I did not know enough about how KSP and Unity GUIs worked to do a good job and settled for an approach based on screen messages. I have learned more since that time and Beaufort is the result. I plan for Beaufort to replace all screen messages in my current mod suite and will be updating those over the next week or two after rolling out Beaufort and addressing any issues which may arise.

**Installation**

Requires but is not bundled with WindAPI. Remove old versions prior to updating. Extract and merge GameData with your existing folder.
